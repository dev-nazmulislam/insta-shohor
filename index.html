Question-01: How does javaScript work?

JavaScript is high-level, often, single-threaded, just-in-time compiled, synchronous and multi-paradigm programming language. Which means it can handle one task at a time and execute line by line. But there have some functionality which is asynchronous. to maintain this asynchronous task it has a single call stack. this call stack is a data structure which racord which task run fast and this call stack calls multiple functions. if one task end then calls next function.
